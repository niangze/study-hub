# StudyHub - 线上答疑互助平台

Java程序设计课程项目。一个面向高校学生的线上答疑互助平台，支持提问、回答、资源分享等功能。

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | Vue 3 + Element Plus + Axios |
| 后端 | Spring Boot 3 + MyBatis-Plus + Spring Security |
| 数据库 | MySQL 8.0 |
| 工具 | Maven + Git + Swagger |

## 团队成员（5人）

| 角色 | 职责 |
|------|------|
| 项目经理 | 需求分析、进度把控、Git管理、联调协调 |
| 后端开发A | 用户模块、答疑模块、数据库设计 |
| 后端开发B | 资源模块、后台管理、数据统计 |
| 前端开发 | 全部前端页面、组件封装、接口对接 |
| 测试/文档 | 测试用例、Bug跟踪、技术文档 |
